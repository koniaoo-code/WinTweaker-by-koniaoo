## 📥 Как установить (просто)

1. Нажми **[⬇️ СКАЧАТЬ](../../releases/latest)**.
2. Скачай **`WinTweaker.exe`**.
3. Запусти его — всё, готово. Ничего ставить не нужно.

> Программа сама попросит права администратора — нажми «Да» (это нужно для настроек системы).
> Подходит для Windows 10 и 11.

*Если хочешь с ярлыком на рабочем столе — скачай `WinTweaker_Setup.exe` (установщик).*

## ✨ Что умеет

- ⚡ **Ускорение** — питание, отключение лишних служб, визуальные эффекты, таймер, приоритеты CPU.
- 🎮 **Игры** — Game DVR, акселерация мыши, полноэкранная оптимизация, сетевой троттлинг, приоритеты MMCSS.
- 🗂 **Проводник** — «Этот компьютер», панель задач, секунды в часах, компактный вид, быстрый доступ.
- 🔋 **Питание** — схема Ultimate Performance, USB suspend, тайм-ауты экрана/сна/диска.
- 🔒 **Приватность** — отключает слежку Microsoft, Cortana, телеметрию, геолокацию, доступ приложений.
- 🛡 **Безопасность** — SMBv1, LLMNR, автозапуск, удалённый помощник, UAC, SmartScreen.
- 🚫 **Без рекламы** — убирает рекламу из меню Пуск, экрана блокировки, Параметров, Проводника и Edge.
- 🧹 **Очистка** — временные файлы, кэш шрифтов/значков, дампы, корзина, журналы в один клик.
- 🌐 **Сеть** — DNS (Google/Cloudflare/AdGuard/Quad9), Teredo, задержки в играх.
- 💻 **Дашборд и Инфо о ПК** — здоровье системы, живой мониторинг, характеристики, бенчмарк CPU.
- 🗑 **Удаление ненужных приложений** (UWP/bloatware), автозагрузка, службы.
- 🌍 Русский и English, тёмная/светлая тема, плавные анимации, пресеты в один клик.

Все изменения **обратимы** — любой твик можно выключить переключателем.
