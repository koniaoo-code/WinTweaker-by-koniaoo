# ⚡ WinTweaker — by koniaoo

> Твикер для Windows: производительность, телеметрия, реклама, очистка.

---

## 🚀 Быстрый старт

1. Скачай все файлы в **одну папку**
2. Дважды кликни **`build.bat`**
3. Он сам установит `customtkinter` + `pillow` + `pyinstaller`
4. Готовый **`WinTweaker.exe`** появится в папке **`dist\`**
5. Запускай `WinTweaker.exe` **от имени Администратора**

---

## 📋 Требования

- Windows 10 / 11
- Python 3.10+ (скачать с [python.org](https://python.org))
- Права администратора для применения твиков

## 🔧 Разделы

### ⚡ Производительность
| Твик | Что делает |
|---|---|
| Максимальная производительность питания | Схема питания «Максимальная производительность» |
| Отключить SysMain (Superfetch) | Освобождает ОЗУ |
| Отключить индексирование поиска | Снижает нагрузку на диск |
| Отключить визуальные эффекты | Убирает анимации и тени |
| Включить Game Mode | Оптимизирует ресурсы под игры |
| Быстрый запуск Windows | Ускоряет загрузку |
| Высокое разрешение таймера | Снижает задержки |
| Отключить патч Meltdown/Spectre | Прирост FPS (⚠ снижает безопасность) |

### 🔒 Телеметрия
| Твик | Что делает |
|---|---|
| Отключить телеметрию Windows | Блокирует отправку данных в MS |
| Отключить историю активности | MS не хранит твои действия |
| Отключить Cortana | Деактивирует голосового ассистента |
| Отключить геолокацию | Запрещает доступ к местоположению |
| Отключить отчёты об ошибках | Не отправляет crash-отчёты |
| Отключить рекламный ID | Запрещает уникальный рекламный идентификатор |
| Отключить запросы обратной связи | Убирает опросы Microsoft |
| Отключить CompatTelRunner | Останавливает сборщики совместимости |

### 🚫 Реклама
| Твик | Что делает |
|---|---|
| Убрать рекламу в меню Пуск | Отключает предложения приложений |
| Убрать рекламу на экране блокировки | Отключает Windows Spotlight |
| Отключить советы Windows | Убирает всплывающие подсказки |
| Отключить автоустановку приложений | Запрещает bloatware |
| Убрать Bing из поиска | Поиск без интернета |
| Отключить рекламу в Edge | Убирает плитки на новой вкладке |

### 🧹 Очистка
| Твик | Что делает |
|---|---|
| Очистить временные файлы | %TEMP% и C:\Windows\Temp |
| Очистить Prefetch | Кэш предзагрузки |
| Сбросить DNS-кэш | Проблемы с сетью |
| Очистить корзину | Все файлы из корзины |
| Очистить журналы событий | Системные логи |
| Очистить кэш миниатюр | Превью файлов Проводника |
| Очистить кэш обновлений Windows | SoftwareDistribution\Download |

## 💾 Настройки

Все переключатели сохраняются в **`wintweaker_settings.json`** рядом с `.exe`.  
При следующем запуске всё восстановится автоматически.
