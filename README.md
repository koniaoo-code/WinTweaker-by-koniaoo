## 📥 Как установить (просто)

1. Нажми **[⬇️ СКАЧАТЬ](../../releases/latest)**.
2. Скачай **`WinTweaker.exe`**.
3. Запусти его — всё, готово. Ничего ставить не нужно.

> Программа сама попросит права администратора — нажми «Да» (это нужно для настроек системы).
> Подходит для Windows 10 и 11.

*Если хочешь с ярлыком на рабочем столе — скачай `WinTweaker_Setup.exe` (установщик).*

---

## ✨ Что умеет

- ⚡ **Ускорение** — питание, отключение лишних служб, визуальные эффекты, Game Mode.
- 🔒 **Приватность** — отключает слежку Microsoft, Cortana, телеметрию, геолокацию.
- 🚫 **Без рекламы** — убирает рекламу из меню Пуск, экрана блокировки, Edge.
- 🧹 **Очистка** — временные файлы, кэш, корзина, журналы в один клик.
- 🌐 **Сеть** — ускорение интернета и снижение задержек в играх.
- 💻 **Инфо о ПК** — характеристики компьютера в красивом виде.
- 🗑 **Удаление ненужных приложений** (UWP/bloatware).
- 🌍 Русский и English, тёмная тема, плавные анимации.

Все изменения **обратимы** — любой твик можно выключить переключателем.
